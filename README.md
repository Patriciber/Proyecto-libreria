# Lumière - Biblioteca Digital Premium

Lumière es una plataforma de lectura digital diseñada con una estética moderna y premium. Ofrece un catálogo diverso de libros, un sistema de carrito de compras y una experiencia de lectura personalizada.

## Características Principales
- **Catálogo Inteligente**: Filtrado dinámico por géneros, precios (Gratis/Premium/Pago) y búsqueda en tiempo real.
- **Vista Previa de Lectura**: Lector inmersivo con ajustes de tipografía y temas visuales (Claro, Sepia, Oscuro).
- **Diseño Premium**: Interfaz responsive con desenfoques, degradados y micro-animaciones.
- **Resiliencia de Datos**: Carga automática de un catálogo de respaldo si el backend no está disponible.

## Tecnologías
- **Frontend**: HTML5, Vanilla CSS3 (Custom Properties), JavaScript (ES6+).
- **Backend**: FastAPI (Python), PostgreSQL (opcional).
- **Diseño**: Remix Icons, Google Fonts (Outfit).
